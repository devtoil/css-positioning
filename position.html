Absolute Positioning

Coordinate System

Absolutely positioned elements base their position off the nearest relatively or absolutely positioned parent.

50% 50%

When a style of top: 50% left: 50% is applied we can see the top left corner of the box being dead center of its parent. Notice that the center of the box is not aligned with the center of its parent.

Dead Center

If we want the center of the box to be dead center of its parent we will need to offset it by half of its width and height. ( here I'm using the css function calc() to achieve this. ex. calc(50% - 50px) )

Relative Positioning

The first box is relatively positioned. Notice how the second box displays in the flow of the page and is not effected by the first box.

Change the positiong of the boxes to absolute using Chrome Dev tools to get a better understanding of the differences.
